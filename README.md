# Geografía Mundialista 2026

App educativa interactiva sobre los 48 países participantes del Mundial 2026.

## Estructura

```
proyecto-mundial-2026/
├── index.html          # Mapa interactivo
├── agregar-ficha.html  # Formulario alumnos
├── panel-profesor.html # Panel validación
├── firebase.json       # Config hosting
├── .gitignore
├── css/
│   └── styles.css
├── js/
│   ├── config-firebase.js
│   ├── mapa.js
│   ├── ficha.js
│   └── profesor.js
└── data/
    └── paises.json
```
